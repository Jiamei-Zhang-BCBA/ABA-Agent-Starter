- **环境**：咨询室桌面 + 自然观察（自由游戏 20 分钟）

## 🎯 Level 2（18-30m 月龄）得分矩阵（满分 85）

| 领域 | Level 2 得分 | 备注 |
|:---|:---|:---|
| Mand（要求） | **1.5 / 5** | 仅在高动机下（想要公主贴纸）自发说 "要贴贴"；绝大多数想要物品仍通过抓手牵引 |
| Tact（命名） | **4 / 5** | 数字、颜色、动物、小马宝莉人物命名非常强；仅复杂情境（多物品并列）会卡壳 |
| Listener（听者） | **3 / 5** | 两步指令 60%；"去拿你的水杯"完成；"把鞋子放进鞋柜然后去洗手"漏第二步 |
| VP-MTS（视觉配对） | **4.5 / 5** | 非常突出：公主 figurine 按颜色分组 100% 正确；数字 1-10 顺序排列独立 |
| Independent Play（独立游戏） | **3 / 5** | 能独立玩贴纸 / figurine 排队 15 分钟；不会假装性游戏（用香蕉当电话） |
| Social Behavior（社交） | **1 / 5** | 对陌生人第一反应躲妈妈身后；对同龄女孩盯着看但不接近 |
| Imitation（模仿） | **3.5 / 5** | 粗大动作模仿（拍手、跺脚）可；精细动作（手指按数）漏细节 |
| Echoic（仿说） | **4.5 / 5** | 能仿整段动画对白；声母辨识度高；仅 zh/ch/sh 混淆 |
| Spontaneous Vocal（主动发声） | **1.5 / 5** | 除了仿说动画，极少自发说话；一天约 20-30 句主动语言 |
| LRFFC（听者命名+特征） | **2 / 5** | "给我红色的" OK；"给我能吃的" 不能泛化 |
| Intraverbal（互动对话） | **0.5 / 5** | 几乎全无；唯一能答的 Intraverbal 是 "小马宝莉主角叫..." → "紫悦" |
| Classroom Routine（课堂常规） | **1.5 / 5** | 圆圈时间坐不住 ≥ 2 分钟；集合排队会离队；但有熟悉物时（贴纸）能坐定 |

**Level 2 总分**：**30.5 / 85** ≈ 36%

## 🚨 Barrier（障碍）评估（满分 24 × 5 = 120，分数越低越无障碍）

| 障碍 | 得分 | 说明 |
|:---|:---|:---|
| 1. Behavior Problem（问题行为） | **2 / 4** | 啃手指每天 20+ 次（焦虑型自我刺激）；离开 preferred 活动哭闹 3-5 分钟 |
| 2. Instructional Control（指令控制） | **2 / 4** | 喜欢的人（妈妈、胡老师）可以 follow；陌生指令者（新老师）会忽略 |
| 4. Defective Mand（Mand 缺陷） | **4 / 4** | 严重缺陷：只有 2-3 种自发 Mand |
| 7. Pronoun Reversal（代词反转） | **4 / 4** | **严重**：I/You 完全反置（"你喝水水"= 我要喝水） |
| 9. Echolalia（回声式言语） | **2 / 4** | 中度：能整段复述动画但不一定情境相关 |
| 10. Defective Intraverbal（互动缺陷） | **4 / 4** | 严重：基本无对答能力 |
| 16. Sensory Defensiveness（感官防御） | **2 / 4** | 中度：怕打雷、鞭炮；抗拒湿黏（橡皮泥）|
| 17. OCD / 排序强迫 | **3 / 4** | 公主/小马 figurine 必须按大小排列；被打断会大哭 |
| 其他 Barriers | 0-1 | 其余未见显著障碍 |

**关键 Barrier 总结**：
1. **代词反转（4/4 极严重）** — 全面影响所有对话性技能
2. **Mand 缺陷（4/4）** — 临床优先干预起点
3. **Intraverbal 缺陷（4/4）** — 需先建 Mand/Tact 基础
4. **问题行为（2/4）** — 啃手指是自我刺激 / 焦虑缓解，非攻击性
5. **排序强迫（3/4）** — 可用作 teaching opportunity，不是干预障碍

## 🎯 评估结论（BCBA 临床解读）

**优势轮廓**：
- 视觉 / 配对能力突出（VP-MTS 4.5）
- 数字 / 颜色 tact 精准（Tact 4）
- 仿说能力强（Echoic 4.5 — 但有反噬风险变回声式）
- 配合度在熟悉人面前良好

**核心缺陷轮廓**：
- **代词反转**（Pronoun Reversal） — 全 Barrier 最严重之一，IEP 必列
- **Mand 几乎缺失** — 从"抓手牵引"到"自发说"需要系统建立
- **Intraverbal = 0** — 所有对答性社交能力都被这个堵住了
- **社交主动性 = 1/5** — 女孩特有的 "看但不接近" 模式

**Level 升级判断**：Level 1 估计已达 ~120/170（相当扎实）；Level 2 还有 55 分待打；Level 3 暂未评估。**临床年龄相当于 24-30 个月**，但数字/仿说能力跨越到 Level 3 水平。

**问题行为功能假设（待 FBA 确认）**：
- **啃手指** → 自我刺激 / 焦虑缓解（自动强化），陌生情境加剧
- **重复提问** → Intraverbal 缺陷的代偿性行为？或焦虑管理？
- **转换困难** → 对 preferred 活动的过度依恋 + Intraverbal 缺陷导致无法理解"5 分钟后停"

## 📌 下游建议

1. **IEP 优先方向**：Mand（主动要求常见物品）→ 代词 I/You 辨识 → 简单 Intraverbal
2. **BIP 方向**：啃手指替代行为（触觉玩具 / 冰粒握持）+ 妈妈焦虑干预
3. **强化物评估**：先测公主贴纸 / 小马 figurine / 数字卡饱和度
4. **教学模式**：PT（Pivotal Treatment）> DTT，利用孩子 preferred 的排序强化 Mand
